# 查询工作表节点模板

```
查询 [目标表]
  筛选: 当 [目标表].[字段] = [当前记录].[字段]
    动态值为空时: 忽略该条件                 ← 公理5
    无筛选结果时: 继续执行                   ← 公理5
  排序: 创建时间 最新的在前
  获取方式: 从工作表获取/从关联记录获取       ← 公理1: 精确查找
  无数据时: 继续执行                         ← 公理5
  查询到多条时: 获取第一条/[需确认]
```

## 约束

- MUST: 按「记录ID=流程参数[拼音缩写+id]」精确查找（公理1）
- MUST: 未获取到数据→继续执行之后节点（公理5）
- MUST NOT: 模糊条件查询（公理1）
- MUST NOT: 未获取到数据时中止流程（公理5）
