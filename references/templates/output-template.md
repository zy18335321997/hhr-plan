#### 格式模板

```
T0 ⏎ 触发节点
    工作表: [表名] | [触发方式]
    │
    ▼
T1 ◇ 分支 (排他/并行)
    │
    ├── 条件A = 值 ────────── 进入路径A
    │       │
    │       ▼
    │   T2 [节点类型] [描述]
    │       [配置项]: [值]
    │       [配置项]: [值]
    │       │
    │       ▼
    │      ...
    │
    └── 条件A ≠ 值
            │
            ▼
          [结束 — 说明]
```

#### 节点符号

```
⏎ 触发           ◇ 分支            ✓ 审批
🔍 获取单条数据    🔍🔍 获取多条数据   ✚ 新增记录
✎ 更新记录        ✉ 通知            ⚙ 子流程
```

#### 各节点标注内容

**触发**:
```
T0 ⏎ 工作表事件触发
    工作表: [表名] | 新增记录触发/编辑记录触发/删除记录触发
```

**分支**:
```
T1 ◇ 分支 (排他)    ← 排他=只走第一条满足的
T1 ◇ 分支 (并行)    ← 并行=同时走所有分支

分支线上写条件，右侧写路径含义:
    ├── 报销类型 = 科目费用 ────────── 进入审批
    └── 报销类型 ≠ 科目费用
```

**获取单条数据** (🔍 typeId=6):
```
T3 🔍 获取单条数据 → 获取关联记录
    从 [触发表].[关联字段] 取 1 条 [目标表] 记录

T3 🔍 获取单条数据 → 查询工作表
    从 [目标表] 筛选取 1 条
    筛选: [字段] = [值]
    排序: [字段] 升序/降序
    无数据时: 继续执行 / 中止
```

**获取多条数据** (🔍🔍 typeId=7):
```
T3 🔍🔍 获取多条数据 → 获取关联记录
    从 [触发表].[关联字段] 取全部 [目标表] 记录 (最多 1000)

T3 🔍🔍 获取多条数据 → 查询工作表
    从 [目标表] 筛选取全部匹配记录
    筛选: [字段] = [值]
    排序: [字段] 升序/降序
    取前: [N] 条
    无数据时: 继续执行 / 中止
```

T10 ✚ 新增记录 [目标表] (逐条新增/批量新增)
