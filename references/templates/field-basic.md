# 基础字段配置模板

每个配置项标注来源：[引擎推导] 或 [需确认]。

```
[字段类型] 字段名 (R行C列)
  默认: [值]/当前用户/触发时间/计算值/[需确认]  ← 状态→默认, 日期→触发时, 拥有者→当前用户
  [必填]/[唯一]                                   ← 追溯字段→必填, 编号/名称→唯一
  字数限制: [min]-[max]/[需确认]                  ← 短文本→100, 长文本→2000, 手机号→11
  格式验证: [规则]/[需确认]                       ← 手机→11位, 邮箱→email
  搜索设置: 可搜索/[需确认]                       ← 名称/编号/常用检索→可搜索
  提示: [用户可见引导文字]                        ← 客户语言
  可见: [条件]/始终/[需确认]                      ← 业务状态驱动
  可编辑: [条件]/始终/[需确认]                    ← 已审批→只读, 系统生成→只读
  宽度: 半行/全行/[需确认]                        ← 短字段→半行, 长文本/附件→全行
```

## 字段类型默认值速查

| 字段类型 | 默认宽度 | 默认字数 | 可搜索 | 必填 |
|---------|---------|---------|--------|------|
| 自动编号 | 全行 | - | ✓ | 自动 |
| 文本(名称类) | 半行 | 100 | ✓ | ✓ |
| 文本(描述类) | 全行 | 2000 | - | - |
| 数值 | 半行 | - | - | [需确认] |
| 金额 | 半行+单位 | - | - | [需确认] |
| 手机/电话 | 半行 | 11 | - | [需确认] |
| 日期 | 半行 | - | - | [需确认] |
| 单选 | 半行 | - | - | 状态类✓ 其余[需确认] |
| 成员(拥有者) | 半行 | - | - | ✓ |
| 成员(非拥有者) | 半行 | - | - | [需确认] |
| 附件 | 全行 | - | - | - |

## 始终设默认值的字段

以下字段引擎自动推导默认值：

- 日期 → 触发时间 或 计算值
- 状态 → 默认值（公理5）
- 拥有者 → 当前用户（公理1）
- 编号 → 自动生成（公理3）
