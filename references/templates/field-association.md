# 关联表字段配置模板 (类型 29)

```
[关联表] 字段名 (R行C列)
  → 关联到: [目标表]                        ← 数据血缘方向
  显示字段: [字段1, 字段2...]               ← 工作流需要什么就显示什么
  排序: [字段] [升序/降序]                   ← 创建时间最新在前
  过滤选择范围:
    当 [条件字段]=[动态值]                  ← 如: 使用状态=空闲
    动态值为空时: 忽略该条件                 ← 公理5
    无筛选结果时: 赋空值 继续               ← 公理5
  操作: 允许选择已有✓  允许打开记录✓  允许新增:[需确认]
  查询到多条时: 获取第一条
  未查询到记录时: 赋空值 继续执行           ← 公理5
```

## 关联方向规则（公理4）

- 关联表(类型29) → 单向引用
- Hub表被所有相关表单向引用
- 新增关联前必须检查不产生 DAG 环路
