# 子表配置模板 (类型 34)

## 子表 vs 共用参照表判断

| 条件 | 用什么 |
|------|--------|
| 明细数据只在一个父表内使用 | **子表(34)** |
| 明细被 ≥2 个模块引用 | **关联表(29/31)** + 括号标注父表名 |

判断方法: 设计时问"这份明细以后别的模块会不会用到？" 不会→子表，会→独立建表。

## 子表配置模板

```
[子表] 字段名 (R行C列)
  → 目标表: [子表名]
  子表内字段:
    [类型] 字段名
      默认: [值]  [必填]
      → 关联到: [目标]  显示字段: [...]
  子表默认展开: [需确认] 行
```

## 主子创建顺序（公理1 / 定理5）

- MUST: 先建主记录，后建子记录
- MUST: 子记录创建时回写主记录ID和所有可用关联引用
- MUST NOT: 先建子后建主
