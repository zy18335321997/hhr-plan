# 口语检验

> 输出方案前，将配置术语替换为客户语言。默念一遍，读出来不像跟同事解释就改。

## 替换表

| 配置术语 | 客户语言 |
|---------|---------|
| 关联表 | 从已有的XX中选择 |
| 工作流事件触发 | 当你新增/修改记录时自动执行 |
| 未获取到数据时继续执行 | 如果没找到关联数据，会跳过但不影响其他操作 |
| 子流程逐条执行 | 每一条明细都会单独处理 |
| 审批人为空时由流程拥有者代理 | 如果审批人不在，会自动转给管理员处理 |
| 回写父引用 | 记录上会自动关联到来源 |
| 他表字段 | 引用的XX信息（只读） |
| 关联记录操作 | 同时更新关联的记录 |
| 获取单条数据 | 查找对应的XX |
| 获取多条数据 | 列出所有相关的XX |

## 检查方法

1. 把输出读出来
2. 如果包含表中最左侧的术语 → 替换为右侧的客户语言
3. 如果替换后句子不通顺 → 重新组织语序
4. 如果一句话里出现 3 个以上配置术语 → 拆成多个短句

## 不允许的行为

- 不说"我们需要建一个关联表"
- 不说"配一个工作流"
- 不说"添加一个他表字段"

说"你需要能从已有的客户列表里选择"、"当你保存时会自动"、"这里会引用客户的名称和电话"。
